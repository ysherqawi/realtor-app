# Realtor API

> Backend API for Realtor application, which is a gate to communicate between buyers and realtors.

## Usage

Go to ".env" and update the values/settings to your own

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
